# Commission Calculator App

Simple beginner project for VS Code + Codex Extension.

## Features
- Input sale price
- Input commission rate
- Input broker split
- Output:
  - Agent commission
  - Broker split amount
  - Net income
- Download a PDF summary after calculating

## Run
1. Open this folder in VS Code.
2. Open `index.html` in your browser.
3. Enter numbers and click **Calculate**.
